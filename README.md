🌊 Limnal AI Flood Visualizer

An AI-powered web application that predicts and visualizes urban flood risks using environmental data to support disaster preparedness and smarter city planning.

Features

- Flood risk prediction
- Interactive visualization dashboard
- User authentication
- Data analysis and insights
- Responsive design

Tech Stack

Frontend
- React.js
- Vite
- JavaScript
- CSS

Backend
- Node.js
- Express.js

Database
- MongoDB Atlas

AI
- Python

Project Structure

frontend/
backend/
README.md

Installation

Clone the repository:
git clone https://github.com/POOJIKASRI-V-29/Limnal-ai-flood-visualizer.git


Install dependencies:
cd frontend
npm install
npm run dev

cd backend
npm install
npm start
